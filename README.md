## Prerequisites
[Rust installation](https://rust-lang.org/fr/learn/get-started/)
[Qemu installation](https://www.qemu.org/download/)

## Set Up
```shell
$ cargo install probe-run
```
```shell
$ rustup component add llvm-tools
```
```shell
$ rustup target add thumbv7m-none-eabi
```

## Build
```shell
$ cargo build
```

## Clean
```shell
$ cargo clean
```

## Run
```shell
$ cargo run
```
